# Heart Disease Prediction

A supervised machine learning notebook that trains and compares classification models for predicting heart disease risk from structured health features.

## Workflow

- Load and inspect the dataset
- Prepare features and target values
- Train classification algorithms
- Compare predictions and model performance
- Generate a prediction from supplied feature values

## Technology

Python · pandas · NumPy · scikit-learn · Jupyter Notebook

## Usage

Open the notebook in Jupyter, run the cells in order, and review the preprocessing, training, evaluation, and prediction steps.

> This project is for education and portfolio demonstration only. It is not a medical diagnostic tool.
